# Citadel A.R.M.O.R. Project Scope

A.R.M.O.R. means Autonomous Response Monitoring Operations and Resilience.

Canonical root: ~/citadel-ai/armor

This module is a local-first active defense feature for Citadel-AI. It must begin with read-only visibility and grow in controlled phases: status, scan, vault, Wi-Fi/Bluetooth/USB awareness, gate, countermeasures, lockdown, drill.

Safety law: automatic observation is allowed. Destructive action requires user approval. No hack-back. No automatic deletion. No lockout behavior.
